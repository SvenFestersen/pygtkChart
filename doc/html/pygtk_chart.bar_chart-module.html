draw_rounded_rectangle(context, x, y, width, height, radius=0)

source code 

Draws a rectangle with rounded corners to context. radius specifies the corner radius in px.

Parameters:
  • context (CairoContext) - the context to draw on
  • x (float) - x coordinate of the upper left corner
  • y (float) - y coordinate of the upper left corner
  • width (float) - width of the rectangle in px
  • height (float) - height of the rectangle in px
  • radius (float.) - corner radius in px (default: 0)
