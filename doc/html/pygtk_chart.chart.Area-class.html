This is a base class for classes that represent areas, e.g. the pie_chart.PieArea class and the bar_chart.Bar class.

Properties

The Area class inherits properties from chart_object.ChartObject. Additional properties:

  • name (a unique name for the area, type: string, read only)
  • value (the value of the area, type: float)
  • color (the area's color, type: gtk.gdk.Color)
  • label (a label for the area, type: string)
  • highlighted (set whether the area should be highlighted, type: boolean).
