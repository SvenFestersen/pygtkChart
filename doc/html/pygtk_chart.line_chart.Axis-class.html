This class represents an axis on the line chart.

Properties

The Axis class inherits properties from chart_object.ChartObject. Additional properties:

  • label (a label for the axis, type: string)
  • show-label (sets whether the axis' label should be shown, type: boolean)
  • position (position of the axis, type: an axis position constant)
  • show-tics (sets whether tics should be shown at the axis, type: boolean)
  • show-tic-lables (sets whether labels should be shown at the tics, type: boolean)
  • tic-format-function (a function that is used to format the tic labels, default: str)
  • logarithmic (sets whether the axis should use a logarithmic scale, type: boolean).
